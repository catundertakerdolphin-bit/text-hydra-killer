# text-hydra-killer

Со мной случается постоянно. Пишешь, всё вроде по делу, а к финалу текст превращается в гидру хаоса: лишние головы уже спорят с главной. Создал skill, который помогает увидеть все головы сразу: лишние обрубить, а какие-то скрутить в бараний рог уробороса, пусть дожирают сами себя на глазах у читателя.

Скилл не "улучшает формулировки", а строит карту смысловых связей: где мысль ветвится, какие тезисы сходятся, какие фрагменты висят отдельно и в какой финальный узел всё можно замкнуть.

Работает с черновиком, набором тезисов, новостью или просто кучей заметок. На выходе — Mermaid-граф и подсказки, какие мосты добавить между ветками.

Удобно, когда текст уже есть, но пора бы закончить.

## Что делает

- Диагностирует тип входа: черновик, набор тезисов, новость, реакция, цитата или гибрид.
- Выделяет смысловые узлы: тезисы, сцены, метафоры, факты, примеры, финальные формулы.
- Собирает ветки по функции: социальная, экономическая, эстетическая, драматургическая, историческая и другие.
- Показывает связи между ветками: причина, контраст, пример, мост, аналогия, петля, финальное замыкание.
- Строит Mermaid-граф, где видно, как мысль расходится и куда приходит.
- Подсказывает, где добавить мост, что переставить, что вынести в p.s., а что отрезать.

## Установка

Скопируйте `SKILL.md` в папку навыков Codex:

```bash
mkdir -p ~/.codex/skills/text-hydra-killer
curl -L https://raw.githubusercontent.com/catundertakerdolphin-bit/text-hydra-killer/main/SKILL.md \
  -o ~/.codex/skills/text-hydra-killer/SKILL.md
```

Можно и проще: прикрепить `SKILL.md` в чат вместе с текстом.

## Пример запроса

```text
Используй text-hydra-killer.
Вот черновик. Построй карту смысловых связей, покажи Mermaid-граф,
где ветки сходятся и где нужно замкнуть финал.
```

## Что получается на выходе

```text
1. Диагноз входа
2. Центральные кандидаты
3. Ветки
4. Матрица связей
5. Mermaid-граф
6. Где ветки сходятся / расходятся
7. Где замкнуть
8. Готовые мосты
```

## Для чего

Не для полировки стиля. Для момента, когда текст уже вырос, но начал плодить головы.

Скилл помогает увидеть не гладкую версию текста, а его нервную систему.
